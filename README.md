# Cloud Security Posture & Compliance Platform (CSPC)

An enterprise-style cloud security platform that analyzes infrastructure configurations, detects misconfigurations, and calculates organizational risk exposure.

---

## 🚀 Features

- Cloud security misconfiguration scanning engine
- Risk scoring system (Critical → Low)
- Compliance mapping (CIS / NIST / Zero Trust)
- REST API for programmatic access (FastAPI)
- SOC-style dashboard (Streamlit)
- Security visualization and reporting

---

## 🧠 Architecture

This project simulates a real-world Cloud Security Posture Management (CSPM) tool with:

- Backend API layer
- Core security engine
- Risk scoring logic
- Frontend security dashboard

---

## 🛠️ Tech Stack

- Python
- FastAPI
- Streamlit
- Pandas
- YAML
- Uvicorn
